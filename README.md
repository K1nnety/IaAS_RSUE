# ИАС Успеваемость РГЭУ РИНХ — Информационно-аналитическая система мониторинга успеваемости

[![Python](https://img.shields.io/badge/Python-3.8+-blue)](https://www.python.org/)
[![Streamlit](https://img.shields.io/badge/Streamlit-1.20+-FF4B4B)](https://streamlit.io/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Веб-приложение на Streamlit для анализа качества образования в учебных подразделениях РГЭУ РИНХ (Ростовский государственный экономический университет).  
Система позволяет загружать данные об оценках студентов из Excel, генерировать шаблоны и заполненные примеры, а также визуализировать ключевые показатели успеваемости.

**Основные возможности:**
- Генерация пустых шаблонов для ручного заполнения преподавателями
- Генерация примера заполненных данных (имитация реальных оценок)
- Просмотр и фильтрация исходных данных
- Расчёт ключевых метрик успеваемости (средний балл, качество обучения, динамика)
- Выявление групп с падением успеваемости
- Персональный анализ траектории студента

## Требования к данным (структура Excel-файла)

Файл должен быть в формате `.xlsx` с **строго фиксированными названиями столбцов** (без лишних пробелов, регистр важен!):

| Столбец          | Тип данных       | Описание                                                                 | Пример значения                     | Обязательный? |
|------------------|------------------|--------------------------------------------------------------------------|-------------------------------------|---------------|
| Факультет       | Строка           | Полное название направления подготовки                                   | "Прикладная информатика"           | Да            |
| Учебный_год     | Строка           | Учебный год в формате "YYYY/YYYY+1"                                      | "2025/2026"                        | Да            |
| Курс            | Целое число      | Номер курса (1–4 или 1–5)                                                | 3                                   | Да            |
| Группа          | Строка           | Код группы (обычно направление-код курса-номер группы)                   | "ПИ-401"                           | Да            |
| Студент         | Строка           | ФИО студента в формате "Фамилия И.О."                                    | "Иванов И.И."                      | Да            |
| Предмет         | Строка           | Название дисциплины                                                      | "Базы данных"                      | Да            |
| Итоговая_оценка | Целое число      | Балл по 100-балльной шкале (обычно 40–100, ниже — неуд/отработка)       | 87                                  | Да            |

**Рекомендации по заполнению:**
- Оценки — целые числа (не дробные).
- Учебный год всегда в формате "2025/2026" (с косой чертой).
- Группы лучше унифицировать (например, ПИ-101, ПИ-102, ИБ-201 и т.д.).
- Пустые ячейки или неверные типы данных могут привести к ошибкам при загрузке.

**Генерация шаблона:**
В приложении есть кнопка «Создать пустой шаблон» → скачивается файл `template.xlsx` с готовыми заголовками столбцов.
